# 👑 SYAPA KING INBOX AUTOMATION

## 🚀 Features
- Facebook Auto Login (Email/Password)
- E2EE Inbox Message Automation
- TXT File Upload for Messages
- Custom Hater Name Prefix
- Adjustable Message Delay
- Live Royal Console
- Key Approval System with WhatsApp
- Admin Panel for Key Management
- Royal VIP Golden-Purple Theme

- Streamlit
- Selenium (Headless Chrome)
- SQLite
- Cryptography
